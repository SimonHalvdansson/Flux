# Flux

This is a work-in-progress Android app whose main purpose is to provide a widget with current and future spotprices on the European electricity market. For data we use a <a href="https://github.com/SimonHalvdansson/Entsoe-Mirror">mirror of the ENTSOE</a> which I've set up in a separate repository.
